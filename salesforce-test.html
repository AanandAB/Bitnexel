🔗 Salesforce Integration Test Page

Setup Instructions

  1. Create a Zapier account at zapier.com
  2. Create a new Zap with "Webhooks by Zapier" as trigger
  3. Choose "Catch Hook" and copy the webhook URL
  4. Add "Salesforce" as action and choose "Create Lead"
  5. Map the webhook payload fields to Salesforce Lead fields
  6. Replace the webhook URL in the code below with your actual Zapier webhook URL
  7. Test this form to verify integration
